## Выполненная работа:


#### Графика:
-Настроил освещение, запёк Reflection Probe <br />
-Настроил Postprocessing <br />

#### Игрок:
-Добавил возможность передвижения вперёд-назад по касанию левого DPad для возможности тестирования газоанализатора. <br />
-Добавил модели рук для игрока <br />
-Добавил скрипт для нажатия пальцем кнопок. Касание регистрируется, если палец касается кнопки и зажат курок Grip. При нажатии вокруг кнопки появится зеленый индикатор. <br />
-Добавил AudioListener на камеру <br />
-Добавил Canvas внутрь камеры <br />

#### Кран-балка:
-Исходные габариты балки больше габаритов вложенных в неё объектов в 12 раз и, в случае, если передвигать узлы кран-балки локально, между ними будет ровно такой же разрыв в скорости передвижения. Это можно компенсировать настройкой скорости, но, если потребуется точное перемещение в настоящих метрах, то это будет проблемой, поэтому вместо локальных координат я взаимодействую с глобальными. <br />
-Перемещение плавное, реализовано через Vector3.MoveTowards() <br />
-Каждая ось управляется отдельным скриптом ...CranController, в котором задаются параметры перемещения по каждой оси - направление, скорость максимальное расстояние, проигрываемый звук. <br />
-При перемещении издаётся звук, предусмотрено 3 варианта для каждой из осей. <br />
-На динамической веревке свисает пульт, который можно брать в руки. На пульте есть набор кнопок, которые нажимаются методом, описанным ранее, и, каждая кнопка подписана через TMPro 3D. <br />
-Методы передвижения и остановки в скриптах осей подписаны на соответствующие UnityEvent'ы контроллера пульта управления <br />
-Далеко от места крепления провода к балке лучше не оставаться, он начинает физически тянуть за собой пульт и бороться с вашей рукой за удержание пульта. Можно пофиксить, высвобождая пульт из руки при достижении определённого расстояния <br />


#### Газоанализатор и опасные зоны:
-Газоанализатор размещён на пьедестале справа от точки спавна <br />
-Газоанализатор состоит из двух объектов (основной блок и зонд), соединённых динамическим проводом <br />
-Измерение расстояния производится от конца зонда до центра зоны, если усложнять и делать расстояние до края зоны, то можно использовать raycast до найденного центра <br />
-На сцене может быть любое количество зон, газоанализатор показывает расстояние до центра ближайшей <br />
-Зоны - копии префаба, контролируемые менеджером зон. Для добавления зоны - достаточно перетянуть новый префаб, она сама добавится в менеджер. <br />
-Для включения и выключения газоанализатора три секунды удерживайте кнопку посередине. При переключении состояния издастся писк, при выключении газоанализатор показывает шкалу длительности удерживания кнопки <br />
